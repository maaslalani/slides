# Code blocks

Slides allows you to execute code blocks directly inside your slides!

Just press `ctrl+e` and the result of the code block will be displayed as virtual
text in your slides.

Currently supported languages:
* `bash`
* `go`
* `ruby`
* `python`
* `elixir`

---

### Bash

```bash
ls
```

---

### Go

```go
package main

import "fmt"

func main() {
  fmt.Println("Hello, world!")
}
```

---

### Ruby

```ruby
puts "Hello, world!"
```

---

### Python

```python
print("Hello, world!")
```

---

### Elixir

```elixir
IO.puts "Hello, world!"
```
